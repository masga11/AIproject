# AI Debate Arena

Веб-приложение, где два ИИ-агента (Философ и Скептик) автономно спорят на заданную тему в несколько раундов.

## Быстрый старт (Ollama — без регистрации)

1. Установите [Ollama](https://ollama.com/download) для Windows.
2. Откройте терминал и скачайте модель:

```bash
ollama pull llama3.2:3b
```

3. Установите зависимости проекта:

```bash
npm install
```

4. Запустите:

```bash
npm run dev
```

5. Откройте [http://localhost:5173](http://localhost:5173)

Ollama работает **локально на вашем ПК** — аккаунты, ключи и лимиты не нужны.

## Альтернатива: Groq (облако)

Если Groq заработает, в `.env`:

```bash
LLM_PROVIDER=groq
GROQ_API_KEY=gsk_ваш_ключ
```

Ключ: [console.groq.com](https://console.groq.com) → **Sign up** (не Sign in!) → через Google или GitHub.

Если видите «does not belong to any organizations» — вы нажали **Sign in** вместо **Sign up**. Нужна именно регистрация нового аккаунта.

## Возможности

- **2 агента** — Философ и Скептик спорят по очереди
- **Настройка раундов** — от 1 до 5 на экране
- **Судья** — подводит итог и называет победителя
- **Стоп** — прервать спор в любой момент
- **История** — прошлые дебаты сохраняются в браузере

## Как это работает

- **Frontend** (React + Vite) — интерфейс и стрим ответов.
- **Backend** (Express) — оркестрация дебатов.
- **Ollama / Groq** — генерация текста (переключается в `.env`).
- **Debate Engine** — 3 раунда, память между репликами.

## Скрипты

| Команда | Описание |
|---------|----------|
| `npm run dev` | Запуск фронта и бэка |
| `npm run frontend` | Только Vite (порт 5173) |
| `npm run backend` | Только Express (порт 3002) |
| `npm run build` | Сборка фронтенда |

## Модели Ollama (умнее = дольше)

| Модель | Качество | Скорость | Скачать |
|--------|----------|----------|---------|
| `llama3.2:3b` | Базовое | Быстро | уже есть |
| `llama3.1:8b` | Выше | ~30–60 сек/реплика | `ollama pull llama3.1:8b` |
| `qwen2.5:7b` | Выше + русский | ~30–60 сек/реплика | `ollama pull qwen2.5:7b` |

На RTX 3050 Ti (4 ГБ) подойдут **8B/7B**, если готовы ждать. Выбор — в интерфейсе, поле **«Модель»**.

## Настройки (.env)

| Переменная | Описание |
|------------|----------|
| `LLM_PROVIDER` | `ollama` или `groq` |
| `OLLAMA_MODEL` | Модель по умолчанию (`llama3.2:3b`) |
| `GROQ_API_KEY` | Ключ Groq (только для `groq`) |
| `PORT` | Порт бэкенда (по умолчанию 3002) |
