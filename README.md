# AI Debate Arena

Веб-приложение для автономных ИИ-дебатов и турниров. Агенты спорят в несколько раундов, судья подводит итог, глобальная память накапливает знания.

## Быстрый старт (Ollama — без регистрации)

1. Установите [Ollama](https://ollama.com/download)
2. Скачайте модель:

```bash
ollama pull llama3.2:3b
```

3. Установите зависимости и запустите:

```bash
npm install
npm run dev
```

4. Откройте http://localhost:5173

## Возможности

### Дебаты
- **2-5 агентов** спорят в несколько раундов (1-5)
- **Судья** выносит вердикт в конце
- **Стриминг в реальном времени** — ответы появляются по мере генерации
- **Индивидуальные настройки** — отдельная модель и температура для каждого агента
- **Горячие клавиши** — Esc останавливает дебаты

### Турнирный режим 🏆
- Агенты соревнуются в парах, победители проходят дальше
- Визуальная турнирная сетка (bracket)
- Корона чемпиона по итогам турнира
- Стриминг каждого матча в реальном времени

### 9 встроенных агентов
| Агент | Роль |
|-------|------|
| 🧠 Philosopher | Философ и аналитик |
| 🔍 Skeptic | Скептик и критик |
| 🔬 Scientist | Учёный-исследователь |
| 🏛️ Politician | Политик-прагматик |
| 📊 Economist | Экономист-аналитик |
| 🚀 TechnoOptimist | Технооптимист |
| ❤️ Humanist | Гуманист |
| 💰 Капиталист | Сторонник свободного рынка |
| ⚖️ Judge | Беспристрастный судья |

### Кастомные агенты
- Создавайте своих агентов с кастомными промптами и цветами
- GUI для создания, редактирования и удаления

### Глобальная память (RAG + SQLite)
- Сохранение истории дебатов и извлечённых знаний в БД
- Перед началом дебатов система ищет похожие темы и скармливает как контекст
- Рейтинг агентов — сколько дебатов, реплик, побед

### Экспорт / Импорт
- **Экспорт** в JSON или Markdown после каждого дебата
- **Импорт** JSON/MD файлов — импортированные дебаты используются как контекст для будущих дебатов

### Интерфейс
- **Тёмная и светлая тема** с переключателем
- **Мобильная адаптация** — компактный вид на экранах < 640px
- **Emoji аватары** агентов
- **Таймер.elapsed** — показывает время дебатов
- **Прогресс-бар** раундов
- **Текстовая озвучка (TTS)** — кнопка 🔊 у каждой реплики
- **Аналитика** — графики побед, активность за 7 дней, рейтинг агентов
- **Случайная тема** — кнопка 🎲 с 12 заготовками

## Настройки (.env)

| Переменная | Описание |
|------------|----------|
| `LLM_PROVIDER` | `ollama` или `groq` |
| `OLLAMA_MODEL` | Модель по умолчанию (`llama3.2:3b`) |
| `GROQ_API_KEY` | Ключ Groq (только для `groq`) |
| `PORT` | Порт бэкенда (по умолчанию 3002) |

## Модели Ollama

| Модель | Качество | Скачать |

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/health` | Статус сервера |
| GET | `/agents` | Список агентов и моделей |
| GET | `/autonomous-debate-stream` | SSE стрим дебатов |
| POST | `/tournament/start` | Создание турнира |
| GET | `/tournament/:id/stream` | SSE стрим турнира |
| POST | `/memory/import` | Импорт дебатов |
| GET | `/memory/history` | История дебатов |
| GET | `/memory/analytics` | Расширенная аналитика |
| CRUD | `/custom-agents` | Управление кастомными агентами |
