# AI Debate Arena

Веб-приложение для автономных ИИ-дискуссий и турниров. Агенты ведут структурированный диалог в 3 раунда, судья подводит итог, глобальная память накапливает знания.

## Быстрый старт

### Вариант 1: Ollama (бесплатно, локально)

1. Установите [Ollama](https://ollama.com/download)
2. Скачайте модель:
```bash
ollama pull llama3.2:3b
```
3. Запустите:
```bash
npm install
npm run dev
```
4. Откройте http://localhost:5173

### Вариант 2: Groq (облако, бесплатно)

1. Получите ключ на [console.groq.com](https://console.groq.com)
2. Создайте `.env`:
```
LLM_PROVIDER=groq
GROQ_API_KEY=your_key_here
```
3. Запустите: `npm install && npm run dev`

## Возможности

### Структурированные дебаты (3 раунда)
- **Раунд 1 — Открытие позиций**: определения ключевых терминов + 3 аргумента
- **Раунд 2 — Аргументация**: ответы на аргументы оппонента + статистика + каверзный вопрос
- **Раунд 3 — Заключение**: итог дискуссии, обращение к судье
- Агенты занимают **противоположные позиции** (за/против)
- Строгое разделение ролей: каждый агент придерживается своей позиции

### Голосовые агенты (Piper TTS)
- Уникальный голос и характер речи для каждого агента
- Интеграция с [Piper TTS](https://github.com/rhasspy/piper) — локальный качественный TTS
- Автоматический fallback на встроенный Web Speech API
- Профили голосов: rate/pitch для каждого агента

### PDF-экспорт
- Два стиля: **Минимализм** (чистый текст, академический стиль) и **Детальный** (цветные карточки, аватары, градиенты)
- Генерация полностью в браузере (html2pdf.js)
- Формат A4 с типографикой

### Турнирный режим 🏆
- Агенты соревнуются в парах, победители проходят дальше
- Визуальная турнирная сетка (bracket)
- Корона чемпиона по итогам турнира
- **Архив турниров** — просмотр завершённых турниров с replay каждого матча

### Кастомные агенты
- Создавайте своих агентов с кастомными промптами и цветами
- GUI для создания, редактирования и удаления

### Глобальная память (RAG + SQLite)
- Сохранение истории дебатов и извлечённых знаний в БД
- Перед началом дебатов система ищет похожие темы и скармливает как контекст
- Рейтинг агентов — сколько дебатов, реплик, побед

### Экспорт / Импорт
- **Экспорт** в JSON, Markdown или PDF
- **Импорт** JSON/MD файлов — импортированные дебаты используются как контекст

### Интерфейс
- **Тёмная и светлая тема** с переключателем
- **Мобильная адаптация** — компактный вид на экранах < 640px
- **Emoji аватары** агентов
- **Таймер** — показывает время дебатов
- **Прогресс-бар** раундов
- **Аналитика** — графики побед, активность за 7 дней, рейтинг агентов
- **Случайная тема** — кнопка 🎲 с 12 заготовками

## Настройки (.env)

| Переменная | Описание | По умолчанию |

| `OLLAMA_BASE_URL` | URL Ollama сервера | `http://localhost:11434/v1` |
| `GROQ_API_KEY` | Ключ Groq API | — |
| `PORT` | Порт бэкенда | `3002` |
| `PIPER_PATH` | Путь к piper.exe | — |
| `PIPER_MODEL` | Имя модели Piper | `ru_RU-ruslan-medium.onnx` |

## Установка Piper TTS (опционально)

1. Скачайте [Piper](https://github.com/rhasspy/piper/releases) — `piper_windows_amd64.zip`
2. Скачайте русскую модель: [ru_RU-ruslan-medium](https://huggingface.co/rhasspy/piper-voices/tree/main/ru/ru_RU/ruslan/medium)
3. Распакуйте `piper.exe`, `.onnx` и `.onnx.json` в одну папку
4. В `.env` укажите:
```
PIPER_PATH=/mnt/c/путь/к/piper/piper.exe
PIPER_MODEL=ru_RU-ruslan-medium.onnx
```

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/health` | Статус сервера + Piper |
| GET | `/agents` | Список агентов и моделей |
| GET | `/tts?text=...` | Piper TTS → WAV |
| GET | `/autonomous-debate-stream` | SSE стрим дебатов |
| POST | `/tournament/start` | Создание турнира |
| GET | `/tournament/:id/stream` | SSE стрим турнира |
| POST | `/memory/import` | Импорт дебатов |
| GET | `/memory/history` | История дебатов |
| GET | `/memory/tournaments` | История турниров |
| GET | `/memory/analytics` | Расширенная аналитика |
| CRUD | `/custom-agents` | Управление кастомными агентами |
